# ハノイの塔　と　自己相似

ハノイの塔と、ハノイの塔の解放の自己相似性ビューア

## Towers タブ

ハノイの塔を解けます。
キャンバスをドラッグすることで手で動かすことができます。
自動解決のボタンは、コンピュータに解かせます。

## Log タブ

自動解決のログをテキストで表示します。
3本の塔を、左から各A, B, Cとしています。
カッコ内の数字は、プレートの番号です。(上から1, 2...n)

## Graph タブ

自動解決のログをグラフィカルに表示します。

各線分は、右方向はA→C、右上方向はA→Bに対応付けています。 [参考](docs/directions.svg)
線分の長狭はプレートの大きさに対応しています。
